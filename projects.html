Культурная география в информационной среде, персоналия: Николай Гумилёв

Курсовой проект по цифровой гуманитаристике

Базы данных Николай Гумилёв SQLite Python JupyterLab ER-диаграммы ДРАКОН-схемы

Описание

Задача: Создать инструмент для систематизации информации о Николае Гумилёве в цифровом формате.

Решение: Разработана информационная система, включающая базу данных SQLite с ER-диаграммой, программу на Python для CRUD-операций, ДРАКОН-схемы для визуализации и мультимедийные материалы для комплексного представления данных.

Ключевые выводы

  • Освоила принципы проектирования баз данных для гуманитарных исследований
  • Научилась создавать ER-диаграммы и ДРАКОН-схемы
  • Приобрела практические навыки работы с SQLite и Python
  • Разработала полноценную информационную систему для академических целей
  • Сочетание гуманитарного и технического подходов к исследованию

Психолингвистика. Речевое поведение и речевые нарушения

Обучающий курс в ТУИС

Психолингвистика Когнитивная психология Педагогика Логопедия Moodle H5P Интерактивное обучение

Описание

Задача: Создать комплексный курс для формирования целостного представления о психологических механизмах речи.

Решение: Разработан учебный курс, который охватывает психологические основы речи, закономерности её развития и практическое применение в профессиональной деятельности.

Особенности курса

Мультиформатная подача теории:
  • Структурированные HTML-страницы
  • Презентации в H5P
  • Текстовые модули в системе Moodle с пошаговым изложением
Практико-ориентированные задания:
  • Интерактивные тесты H5P
  • Интерактивные упражнения LearningApps
  • Кейс-стади с реальными примерами речевых нарушений
  • Диагностические тесты с автоматической проверкой
  • Творческие задания

Ключевые выводы

  • Структурировала знания о когнитивных основах коммуникации
  • Разработала практические модули для педагогической практики
  • Изучила методы диагностики речевых нарушений у детей
  • Создала систему практических упражнений

Книжный адвент-журнал

Ежемесячные литературные подборки с атмосферным оформлением

Литературная подборка Дизайн полиграфии Видеопроизводство Литературный анализ

Описание

Задача: Читатели часто сталкиваются с проблемой выбора книг, соответствующих сезонному настроению и личному эмоциональному состоянию. Существующие рекомендательные системы слишком обезличены и не учитывают "атмосферность" чтения.

Решение: Создание ежемесячного адвент-журнала с тематическими подборками книг, идеально подходящих для чтения в конкретном месяце.

Особенности проекта

  • Тематические подборки: Книги, подобранные под настроение каждого месяца
  • Атмосферный дизайн: Визуальное оформление, отражающее сезонную тематику
  • Мультиформатность: Печатный/цифровой журнал и видеоконтент
  • Сезонные обзоры: Детальные рекомендации с учетом времени года
